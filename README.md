# Εφαρμογή Καταγραφής Ασθενών Νοσοκομείου

Μικρή web εφαρμογή με PHP και MySQL για καταγραφή, εμφάνιση, αναζήτηση, επεξεργασία και διαγραφή ασθενών νοσοκομείου.

## Λειτουργίες

- Καταχώρηση ασθενή με όνομα, επώνυμο, ΑΜΚΑ, ύψος, βάρος, ημερομηνία γέννησης, ημερομηνία εισαγωγής και αιτία εισαγωγής.
- Αποθήκευση των στοιχείων σε βάση δεδομένων MySQL.
- Προβολή όλων των ασθενών σε πίνακα.
- Αναζήτηση με όνομα, επώνυμο ή ΑΜΚΑ.
- Επεξεργασία στοιχείων ασθενή.
- Διαγραφή ασθενή με επιβεβαίωση.
- Βασικοί έλεγχοι εγκυρότητας για υποχρεωτικά πεδία, ΑΜΚΑ, αριθμητικά πεδία και ημερομηνίες.

## Αρχεία εφαρμογής

- `index.php`: φόρμα νέας καταχώρησης ασθενή.
- `save_patient.php`: αποθήκευση νέου ασθενή στη βάση.
- `patient_list.php`: λίστα, αναζήτηση και ενέργειες ασθενών.
- `edit_patient.php`: φόρμα επεξεργασίας ασθενή.
- `update_patient.php`: ενημέρωση στοιχείων ασθενή.
- `delete_patient.php`: διαγραφή ασθενή.
- `db.php`: σύνδεση με τη βάση MySQL.
- `patients.sql`: δημιουργία βάσης και πίνακα.
- `style.css`: εμφάνιση της εφαρμογής.

## Βάση δεδομένων

Η εφαρμογή χρησιμοποιεί τη βάση `hospital_db` και τον πίνακα `patients`.

Για εγκατάσταση:

1. Ανοίξτε το XAMPP.
2. Εκκινήστε Apache και MySQL.
3. Ανοίξτε το phpMyAdmin.
4. Κάντε import το αρχείο `patients.sql`.

## Εκτέλεση

Με PHP server:

```powershell
C:\xampp\php\php.exe -S 127.0.0.1:8000 -t .
```

Μετά ανοίξτε στον browser:

```text
http://127.0.0.1:8000/index.php
```

## Στοιχεία σύνδεσης

Οι ρυθμίσεις σύνδεσης βρίσκονται στο `db.php`:

```php
$host = 'localhost';
$database = 'hospital_db';
$username = 'root';
$password = '';
```

Αν η MySQL έχει διαφορετικό χρήστη ή κωδικό, αλλάξτε αυτές τις τιμές.
